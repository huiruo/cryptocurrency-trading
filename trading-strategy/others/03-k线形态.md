走势一共有三种，上升趋势、下降趋势和横盘整理
```
1.上涨
当股价突破趋势线，并且一直在趋势线上方运行不有效跌破趋势线，一般有没有趋势，需要上涨一波回调不破趋势线才能说是趋势走好，但是即便是趋势走好那也不能说就不会再次走坏，只是说有趋势的票相对于没趋势的上涨的概率更大；

2.下跌
下跌：股价跌破趋势线，短时间没有反弹回到趋势线上方，一直在趋势线下方运行。

3.横盘
股价围绕着趋势线在某一区间来回震荡，即不向上也不向下；

上涨中的横盘
下跌中横盘
高档横盘
低位横盘

```

### 震荡
```
横盘震荡就是指上证指数在某一个箱体震荡，指数在某一个点位区域徘徊，涨跌互现，通常就是指多空平衡状体被成为股市横盘。经过一波下跌（底部），在横盘期间涨跌幅度很小，涨跌幅度偏大的是箱体震荡。有时规则底部破位要及时止损，由于大势（大盘）向跌，场内资金无心应战，出现破位要注意风险。这种在一个区间来回震荡的横盘很难说是否有主力，但是就在某一天出现放量涨停形成突破，这种上攻意愿暴露了短线机会。
```

```
横盘震荡分为：上涨中的横盘、下跌中横盘、高档横盘、低位横盘。

上涨中的横盘：股票急速上涨之后，需要稍作调整，但是人气和成交量并没有减弱，通常形成旗形技术形态。下跌中横盘：经过一段时间下跌之后，有反弹迹象，但过后依然下跌。投资者可在这个反弹区间找机会离场。高档横盘：股票上涨一段时间之后，没有上升空间，出现滞涨的情况。但是主力又不能立马全部抛售，只能用时间换空间，在一个矩形区间震荡，形成圆弧顶技术形态。股市谚语：盘久必跌说得就是这种情况。低位横盘：股票经过长时间的下跌之后，已经没有下跌空间，但此时又不能立马拉升股票，而是通过慢慢地低吸筹码，在一个箱体区间低位震荡，等待时机上涨，通常形成圆弧底技术形态。
```

1.阴阳数量代表总体趋势

2.实体大小表示内在动力和趋势强弱
K线的“实体大小”是内在动力大小的外在表现，通俗一点说”质量越大，速度越快的物体，起惯性冲击力越大”，实体的长度大小表示上涨或者下跌趋势的强弱，阳线实体越大表明内在上涨动力越充足，其上涨的动力大于实体小的阳线，同理阴线实体越大，下跌动能也越大。

3.影线长短反映转折意愿
在K线实体变化不大的情况下，上影线和下影线越来越长，说明市场震荡幅度越来越大，多空博弈越来越激烈。
