## 4.均价计算
2022.10.07 完成
```
之一次买入数量*买入价格+第二次买入数量*买入价格+第N次买入数量*买入价格+手续费）/（之一次买入数量+第二次买入数量+第N次买入数量

之一次买入股票100股，价格为5元，第二次买入股票200股，价格为4.8元，第三次买入股票300股，价格为4.5元，在不计算手续费的情况下平均成本为：（100*5+200*4.8+300*4.5）/（100+200+300）=4.68元。
```
