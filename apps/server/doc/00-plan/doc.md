## 现货交易手续费
```
https://www.binance.com/zh-CN/support/faq/e85d6e703b874674840122196b89780a

交易手续费将在您接收的资产中扣除。例如，如果您购买以太币/USDT，则手续费以以太币支付。
如果您卖出以太币/USDT，则手续费以USDT支付。

例如：
您以每股3,452.55USDT的价格下单购买10以太币：
交易手续费=10以太币*0.1%=0.01以太币 

或者您以每股3,452.55USDT的价格下单出售10以太币：
交易手续费=（10以太币*3,452.55USDT）*0.1%=34.5255USDT
```

## 使用​​BNB抵扣spot手续费
```
在币安平台上参与交易的用户，无论交易何种代币，在需支付交易手续费时，如持有足额BNB币，系统会对所需支付的手续费进行打折优惠（具体折扣率见下表），并按当时市值折算出等值BNB币数量，使用BNB币完成手续费的支付。
例如，您可享有现货交易和杠杆交易手续费25%的折扣，合约交易手续费10%的折扣。如您是VIP用户，您可以获得更高的手续费折扣。更多介绍请参阅BNB权益介绍。

BNB市值折算逻辑是？
若应收交易手续费为BUSD，则根据市场BUSD与BNB兑换比例进行折算，取BUSD/BTC和BNB/BTC市场最新价，BUSD/BNB的兑换比例=(BUSD/BTC)市场最新价/(BNB/BTC)市场最新价。
```
