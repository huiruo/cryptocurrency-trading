## 获取实时价格，计算盈亏比
```
行情接口-->最新价格：
GET /api/v3/ticker/price

浮动盈亏=（当天结算价－开仓价格）×持仓量×合约单位－手续费


盈亏比例计算：卖出价减买入价之差，再除以买入价。
盈亏比例 = （卖出交割成本 - 买入交割成本） / 买入交割成本 * 100%
例如：
买入1000股10元的股票，11元卖出，则公式计算：
（11 * 1000 - 10 * 1000） / （10 * 1000） * 100%
= （11000 - 10000）/ 10000 * 100%
= 1000 / 10000 * 100%
= 0.1 * 100%
= 10%
得：盈利为10%。
```
